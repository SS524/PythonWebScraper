Application Workflow

1. Run the Application 
2. Application will fetch the details of all the Indian airports 
3. This list of airports will be passed on to the dropdown list of the UI form 
4. User will enter the source, destination and, date from the form and will click on the search button
5. After submitting the form, the application will fetch all the flight details based on the inputs provided
   from a different website.
6. The scraped data will be saved to a cloud based database (Datastax Db)
7. Finally those results will be rendered in a tabular format.
